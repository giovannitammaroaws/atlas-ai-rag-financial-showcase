# 08 - Cost Trade-offs

This view explains the architecture trade-offs behind infrastructure and AI spending.

Main trade-off dimensions:
- security boundary vs monthly fixed cost
- performance/latency vs token and compute cost
- richer retrieval context vs higher inference spend
- operational simplicity vs fine-grained control

Use this section to justify design choices in portfolio reviews and technical discussions.
